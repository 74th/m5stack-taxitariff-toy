- M5Stack Michanical Key Button Unit: https://www.switch-science.com/products/8303

## how to build

### VS Code

1. Open in VS Code
2. Install "PlatformIO IDE" extension
3. Click PlatformIO icon on the left sidebar
4. Click "Upload" button and wait for the build and upload to finish
5. Click "Upload File System Image" button and wait for the build and upload to finish

### CLI

```
pio run --target upload
pio run --target uploadfs
```
